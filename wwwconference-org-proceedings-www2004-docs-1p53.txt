 trust negotiation policies specify which credentials   and other resources   to disclose at a given state of the trust negotiation, and the conditions to disclose them. we will refer to this policy in our examples. trust negotiation policies are interpreted by negotiation controllers. we identify strategies that can be used to manage the negotiation instances when policies are changed. this is done by creating one negotiation controller instance for each policy. it assists developers in specifying and modifying negotiation policies. negotiation strategies control which credentials to disclose, when to disclose them, and when to terminate a negotiation.
